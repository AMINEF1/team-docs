# Standards SimpleRAN

## Standards de référence

SimpleRAN s'appuie sur des standards ouverts et reconnus. Cette page recense les principales références.

---

## 3GPP (Radio Access Network)

### New Radio (5G NR)
- **TS 38.104** : NR Frequency bands and channel bandwidths
- **TS 38.211** : NR Physical channels and modulation
- **TS 38.212** : NR Multiplexing and channel coding
- **TS 38.213** : NR Physical layer procedures for control
- **TS 38.214** : NR Physical layer data processing
- **TS 38.321** : NR MAC protocol
- **TS 38.322** : NR RLC protocol
- **TS 38.323** : NR PDCP protocol
- **TS 38.331** : NR RRC protocol

### Evolved Universal Terrestrial Radio Access (4G LTE)
- **TS 36.104** : LTE Frequency bands and channel bandwidths
- **TS 36.211** : LTE Physical channels and modulation
- **TS 36.212** : LTE Multiplexing and channel coding
- **TS 36.213** : LTE Physical layer procedures for control

### Interfaces
- **TS 38.401** : Architecture description (NG-RAN)
- **TS 38.413** : NG-AP protocol
- **TS 38.425** : F1-AP protocol (DU-CU interface)
- **TS 38.472** : E1-AP protocol (CU-UP to CU-CP)

### Core Network (5GC)
- **TS 23.501** : System Architecture
- **TS 23.502** : Procedures for use of the 5G System
- **TS 29.571** : Common API Framework

---

## ETSI (European Standards)

### Open RAN Architecture
- **GS MEC 003** : Multi-access Edge Computing (MEC) Deployment
- **GR MCON 001** : Network Management
- **TS 102 692** : Lawful Interception

### vRAN Architecture
- **GS NFV 001** : Network Functions Virtualization (NFV)
- **GS NFV-INF 001** : NFV Infrastructure

---

## IETF (Internet Engineering)

### Fronthaul Standards
- **RFC 8572** : YANG Data Model for IEEE 802.1AB (LLDP)
- **RFC 3394** : Advanced Encryption Standard (AES) Key Wrap Algorithm

### eCPRI (evolved Common Public Radio Interface)
- **ORAN-WG4.CUS.0-v08.00** : Fronthaul Working Group Specs
- **ORAN-WG5.CUS.0-v05.00** : Radio Unit Specifications
- **Split 7** : RRH / DU split definition

### Transport
- **RFC 2113** : IP Router Alert Option
- **RFC 9000** : QUIC: A UDP-Based Multiplexed and Secure Transport
- **RFC 5905** : Network Time Protocol Version 4

---

## IEEE

### Wireless Standards
- **802.11ax** : Wi-Fi 6 (Coexistence with 5G)
- **802.11be** : Wi-Fi 7

### Network Management
- **802.1Q** : Virtual LANs (VLANs)
- **802.1p** : QoS and CoS

---

## OAI Specifications

### OpenAirInterface Stack
- **OAI gNB Specifications** : Distributed unit implementation
- **OAI 5G Core** : Network function implementation
- **OAI Module Interfaces** : Inter-module contracts

- **Free5GC Architecture** : NF design and flow

## ORAN Alliance (non-proprietary)

### O-RAN Specifications (Reference)

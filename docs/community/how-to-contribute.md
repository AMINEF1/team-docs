# Comment contribuer

SimpleRAN est un projet ouvert et collaboratif. Voici comment contribuer !

## Contributions bienvenues

### Code
- Pull requests sur GitHub
- Contributions OAI, srsRAN, Free5GC
- Bugfixes et optimisations

### Documentation
- Améliorer cette doc
- Ajouter tutorials
- Traduire en d'autres langues

### Testing
- Valider déploiements
- Tester certifications RRH
- Performance benchmarking

### Governance
- Participer aux décisions
- Proposer des améliorations
- Rejoindre le Libre Endowment Fund

## Pas de NDA requis

**Principe clé** : Zéro NDA pour joindre SimpleRAN. Transparence totale !

## Processus

1. Fork le projet
2. Créer une branche `feature/your-feature`
3. Commit avec messages clairs
4. Push et créer PR
5. Review par la communauté
6. Merge après approbation

## Repository structure

# SimpleRAN team documentation

This site is the shared technical documentation space for the team working on SimpleRAN, 5G core, Open RAN experiments, and lab deployments.

## What belongs here

- Architecture decisions and network diagrams
- Deployment procedures for lab and production-like setups
- Experiment notes with commands, parameters, and results
- Troubleshooting steps for recurring issues
- References for standards, interfaces, and tools
- Team contribution and governance notes

## Main sections

- [Architecture](./architecture/overview.md): global view of the 5G and Open RAN environment
- [Deployment](./deployment/quickstart.md): quickstart, infrastructure, SlapOS, and Wendelin notes
- [Experiments](./experiments/iot-scenarios.md): reusable experiment writeups and scenarios
- [Standards](./api-reference/standards.md): interfaces and reference material
- [Community](./community/how-to-contribute.md): contribution workflow and team rules

## Documentation rule

A useful page should explain the context, the commands or configuration used, the result, and what the next person should do with that information.

For new notes, start with [How to document](./community/how-to-document.md).
